核心功能

每一步都由 AI 驱动

从信息捕获到结构化输出，ClueMind 打造完整的思维升维闭环。

AI 伙伴复盘 + 交互式知识画布 — 同步协作的真实界面
01
捕获与录入
灵感来了？随时随地，一键收入收件箱。
  • 全局快捷键 Ctrl+Shift+D，系统全局可用
  • 轻量级浮层，零摩擦录入
  • 原始收件箱管理未处理的素材
02
AI 伙伴式复盘
不是审问式问答，是真正的思维伙伴。
  • AI 先思考，主动提出框架建议
  • 对话语气自然，像与顾问共创
  • 支持 GPT-4o / Claude / GLM / MiniMax
03
交互式知识画布
可视化构建你的思维地图。
  • 可视化图编辑器，节点 + 边操作
  • 三态生命周期：虚拟 → 已确认 → 已锁定
  • 拖拽式布局，内联编辑所有内容
04
Mindscape 全局星图
一览所有知识架构分区，发现重叠与空白。
  • 全局思维星图，高维度概览
  • 重叠知识用虚线 + 发光标签直观展示
  • 点击任意区域进入上下文 AI 深度对话

隐私设计

你的数据，只属于你

ClueMind 采用本地优先架构，从设计层面保障数据主权。

LP
本地优先存储
所有知识框架、素材和对话历史仅存储在你的设备上，不上云。
SK
系统密钥环
API Key 安全保存在系统密钥环中，不存入任何配置文件。
DC
直连 AI 服务
API 调用直接从你的设备发出，无中间服务器，无代理转发。
0T
零遥测
无遥测、无分析、无追踪、无数据收集。完全静默运行。
